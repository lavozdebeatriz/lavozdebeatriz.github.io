Publica los cambios de la web de lavozdebeatriz en GitHub Pages.

Pasos a seguir:
1. Si el usuario no ha proporcionado un mensaje de commit en $ARGUMENTS, pregúntale: "¿Qué mensaje quieres para el commit?"
2. Ejecuta `git add .` en el directorio del repo (`C:/Users/Usuario/Desktop/Proyectos IA/Proyectos WEB/lavozdebeatriz-repo`)
3. Ejecuta `git commit -m "mensaje"` con el mensaje proporcionado
4. Ejecuta `git push`
5. Confirma que todo ha ido bien y recuerda a Beatriz que en unos segundos estará live en https://lavozdebeatriz.github.io
